# 项目简介

本项目是基于PyTorch框架开发的工业视觉异常检测与基准库。它包含了深度学习模型集合与推理工具，旨在为工业检测领域提供一套高效的视觉异常检测解决方案。功能模块涵盖数据预处理、模型训练、模型评估及推理等。技术特点包括采用卷积神经网络（CNN）和循环神经网络（RNN）等先进算法，支持多种损失函数及优化器选择，助力开发者快速搭建、训练并优化检测模型。同时，项目提供了详尽的文档说明，方便用户了解和使用全部源码及资料。


## 核心功能
- 深度学习模型集合与推理工具
- 数据预处理模块
- 模型训练功能
- 模型评估系统
- 异常检测推理工具
- 详尽的文档说明及全部资料

## 适用场景
- 工业视觉异常检测
- 工业检测领域解决方案搭建
- 检测模型快速部署与优化
- 深度学习模型研究与开发

## 技术特点
- 基于PyTorch框架开发
- 采用卷积神经网络（CNN）算法
- 集成循环神经网络（RNN）技术
- 支持多种损失函数选择
- 提供多种优化器选项
- 完整的工业视觉检测工具链
